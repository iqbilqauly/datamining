#

Deskripsi Data#

Dataset iris adalah kumpulan data yang sering digunakan dalam analisis statistik dan pembelajaran mesin. Dataset ini terdiri dari 150 sampel dari tiga spesies iris (Iris setosa, Iris virginica, dan Iris versicolor), masing-masing memiliki 50 sampel. Untuk setiap sampel, terdapat empat fitur numerik: SepalLength, SepalWidth, PetalLength, dan PetalWidth, yang diukur dalam sentimeter.

Penjelasan Fitur#

  1. Panjang Sepal (sepal length): Ini adalah panjang dari sepal atau bagian yang menonjol dari bunga yang melindungi kelopak. Diukur dalam sentimeter (cm).

  1. Lebar Sepal (sepal width): Ini adalah lebar dari sepal bunga iris. Diukur dalam sentimeter (cm).

  1. Panjang Kelopak (petal length): Ini adalah panjang dari kelopak atau bagian bunga yang berwarna. Diukur dalam sentimeter (cm).

  1. Lebar Kelopak (petal width): Ini adalah lebar dari kelopak bunga iris. Diukur dalam sentimeter (cm).

Deteksi Outliers (Local Outlier Factor)#

Konsep Local Outlier#

Outlier adalah bagian dari data yang sangat berbeda dari sebagian besar data lainnya. Mungkin sulit untuk menentukan apa yang tepat dianggap sebagai outlier, karena bisa bersifat subjektif. Outlier dapat dianggap sebagai bagian data yang sangat rendah atau tinggi, jauh dari rata-rata atau norma dari kumpulan data. Outlier bisa disebabkan oleh kebetulan atau kesalahan dalam pengukuran. Hal ini dapat mempengaruhi hasil analisis data secara keseluruhan, sehingga penting untuk mengidentifikasi dan menghapus outlier dari data. Ada berbagai cara untuk mengidentifikasi outlier, dan tidak ada cara baku yang tepat untuk setiap data. Dalam bagian selanjutnya, akan dibahas 5 cara untuk mendeteksi outlier.

Menghitung outlier menggunakan python(scikit learn)#

Label -1 menunjukkan bahwa sampel dianggap sebagai outlier, sementara label 1 menunjukkan bahwa sampel dianggap sebagai data normal.

from sklearn.neighbors import LocalOutlierFactor

    (0,0,1),

    (1,2,2),

    (0,0,0),
    (0,1,2),
    (3,3,3)

]

lof = LocalOutlierFactor(n_neighbors=3, contamination=0.1)

Data Modelling#

Naive Bayes Gausian#

Naive Bayes Gaussian adalah varian dari algoritma klasifikasi Naive Bayes yang diasumsikan memiliki distribusi Gaussian (normal) untuk setiap kelas. Algoritma ini sering digunakan untuk masalah klasifikasi di mana fitur-fitur numerik atau kontinu terlibat.

Prinsip dasar dari Naive Bayes Gaussian mirip dengan Naive Bayes biasa, namun distribusi fitur-fitur numerik diasumsikan mengikuti distribusi Gaussian. Secara matematis, algoritma ini memprediksi kelas sebuah contoh dengan menghitung probabilitas posterior untuk setiap kelas berdasarkan distribusi Gaussian dari fitur-fiturnya.

Langkah-langkah umum dalam menggunakan Naive Bayes Gaussian adalah sebagai berikut:

  1. Persiapan Data: Persiapkan data yang akan digunakan untuk pelatihan model. Pastikan data telah dibagi menjadi fitur-fitur dan label kelas.

  1. Estimasi Parameter: Hitung rata-rata (mean) dan varians dari setiap fitur untuk setiap kelas dalam data latih. Ini diperlukan untuk mengestimasi distribusi Gaussian.

  1. Klasifikasi: Untuk setiap contoh dalam data uji, hitung probabilitas posterior untuk setiap kelas menggunakan distribusi Gaussian dari fitur-fiturnya. Kelas dengan probabilitas posterior tertinggi dipilih sebagai prediksi model.

  1. Evaluasi: Evaluasi model menggunakan metrik yang sesuai seperti akurasi, presisi, recall, atau F1-score.

Meskipun Naive Bayes Gaussian relatif sederhana dan cepat, namun diasumsikan fitur-fitur dalam data independen satu sama lain, yang mungkin tidak selalu terjadi dalam data nyata. Meskipun demikian, algoritma ini sering kali memberikan hasil yang baik terutama untuk dataset yang cukup besar dan sesuai dengan asumsinya.

Implementasi ke Data Iris secara manual#

1. Prior Probability (P(y))#

Prior Probability adalah probabilitas kemunculan kelas tertentu sebelum mengamati fitur. Dalam contoh Iris Dataset, kita memiliki 3 kelas: Setosa, Versicolor, dan Virginica.

Untuk data training yang digunakan sebagai berikut

Jumlah data setiap kelas yaitu : Setosa = 38 Versicolor = 41 Virginica = 41 Total = 120

Maka, Prior Probability untuk masing-masing kelas adalah: P(Setosa) = 38 / 120 P(Versicolor) = 41 / 120 P(Virginica) = 41 / 120

2. Likelihood (P(x|y))#

Likelihood adalah probabilitas mengamati fitur-fitur tertentu dalam data yang diberikan kelas. Kita menggunakan distribusi Gaussian (normal) untuk menghitung likelihood pada Naive Bayes Gaussian.

Untuk menghitung likelihood maka kita harus hitung rata-rata (μ) dan standar deviasi (σ) dari setiap fitur Sepallength, Petallength, Sepalwidth, Petalwidth pada setiap kelas.

Berikut rata-rata (μ) dan standar deviasi (σ) dari setiap fitur disetiap kelas dan X sebagai data test yang mau diprediksi :

  1. Setosa

  1. Versicolor

  1. Virginica

Kemudian, kita hitung nilai PDF (Probability Density Function) untuk masing-masing fitur dan kelas :

rumus likelihood

  • Setosa

  • Versicolor

  • Virginica

3. Posterior Probability (P(y|x))#

Posterior Probability adalah probabilitas kelas setelah mengamati fitur-fitur dalam data. Untuk setiap sampel data baru, kita hitung probabilitas posterior untuk setiap kelas dan prediksi kelas dengan probabilitas tertinggi.

Rumus posterior

bisa ditulis dengan = “posterior = likelihood x prior/evidence”

  • Setosa

  • Versicolor

  • Virginica

Kemudian, kita dapat membandingkan ketiga probabilitas posterior tersebut dan memilih kelas dengan probabilitas tertinggi sebagai prediksi. Nilai posterior tertinggi dari semua kelas adalah dari kelas setosa maka data test yang dijadikan testing termasuk ke kelas setosa.

import numpy as np
import matplotlib.pyplot as plt
import pandas as pd

dataset = pd.read_csv('https://raw.githubusercontent.com/mk-gurucharan/Classification/master/IrisDataset.csv')
X = dataset.iloc[:,:4].values
y = dataset['species'].values
dataset.head(5)

from sklearn.model_selection import train_test_split
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size = 0.2)


from sklearn.preprocessing import StandardScaler
sc = StandardScaler()
X_train = sc.fit_transform(X_train)
X_test = sc.transform(X_test)

from sklearn.naive_bayes import GaussianNB
classifier = GaussianNB()
classifier.fit(X_train, y_train)

y_pred = classifier.predict(X_test) 
y_pred

from sklearn.metrics import confusion_matrix
cm = confusion_matrix(y_test, y_pred)

from sklearn.metrics import accuracy_score 
print ("Accuracy : ", accuracy_score(y_test, y_pred))
